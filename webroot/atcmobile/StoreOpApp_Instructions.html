ATC Store Operations - Login Instructions

To login and use ATC Store Operations, You need to install
ATC Store Admin, then register your Store Owner account, add catalog/items, store's staff accounts.
The store staff accounts for ATC Store Operations login must be appropriate functions enabled.

Following are steps to get ATC Store Operations staff login account:

  1. Install ATC Store Admin app.

  2. Register Store Owner account in ATC Store Admin app. Then login into Store Admin with your Store Owner account.

  3. Add your stores in Store Owner account.

  4. Go to Store's staffs list, add more staffs for your store:



  5. With each store's staff, you need to check/change Access codes:


    "Store Code", "User Code" and "Pass Code" fields will be used to login into ATC Store Function apps like ATC Store Operations

  6. Assign functions to Staffs:


    For staffs on Store Operations line, you need to activate/enable so that they can login into ATC Store Operations.
  7. You can print staff login card:

    this login card can be used to login into ATC Store Operations

  8. After you have staff login credentials for Store Operations, go back to ATC Store Operations




  9. Login into ATC Store Operations with your printed login card for that staff:


  10. Select your Store Operations function line:


  11. ATC Store Operations screens:
